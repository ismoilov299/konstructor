FAQ

DxBot.ru является сервисом для автоматического создания ботов.

Мы создаём ботов совершенно бесплатно.

В данный момент создать можно: 1. Реферальный Bot 2. Онлайн Кинотеатр 3. Bot Знакомств 4. ChatGPT 5. All Save бот

В каждом боте нашего сервиса есть кнопка "заработать", которая ведёт в продуманную реф. программу. Пользователи будут зарабатывать приглашая друзей в бот.

При запуске бота можно добавить несколько каналов-спонсоров. Администраторы каналов платят вам за подписчиков, а вы делитесь прибылью с реферами, которые участвуют в реф. системе.

Для начала нужно выбрать шаблон бота. Далее переходим в @BotFather > /start > /newbot > Ведите название > Введите никнейм на латинице и с окончанием "bot". В конце вам придет сообщение с Token HTTP API( пример: 9732158079:AAGTbmZJJ9bU-fJTVU60FV81t7vVWLsHkkE ). Нужно скопировать токен и вставить его в поле регистрации бота на сайте.
